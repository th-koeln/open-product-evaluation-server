Open Product Evaluation

Open Product Evaluation is a felxible, multi-purpose Evaluation Platform. It provides a rich feautreset to build customized evaluation experiences. Including a full-fledged administration dashboard.

Github Read docs

Whats so special about OPE?

Here's what OPE has to offer.

Speed

OPE accelerates the entire evaluation process. From the creation to the execution and evaluation of the evaluation.

Flexibility

There are no limits to the devices that can be used for evaluation. The evaluation can be performed on tablets, terminals, smartphones of the participants or any other devices.

Reach

The evaluation can take place anywhere. At home, on the road or anywhere else. This allows you to obtain feedback from many different people and thus obtain a representative opinion.

Dashboard

OPE inculdes a extensive web dashboard. It features an survey editor, user management and visualisations of the survey results. Keep track of all your surveys, devices, and domains.

Extensive real-time capabilities

OPE is build to support instant communication amongst all devices. See the survey results change in real time.

Extendable

Build yout onw Cient from scratch or use our reference implementation as a starting point.

Open Source
